# Parent User Manual

## Getting Started

Welcome to the Parent Portal. Monitor your children's academic progress, attendance, and communicate with teachers.

## Logging In

1. Navigate to your college portal
2. Click **Login**
3. Enter provided username and password
4. Access Parent Dashboard

## Dashboard

The dashboard displays:
- **All Children**: Cards for each child
- **Overall Stats**: Combined statistics
- **Recent Communications**: Message history
- **Important Alerts**: Attendance/performance warnings

### Per Child Summary
Each child's card shows:
- Attendance percentage
- Average grades
- Pending assignments
- Pending fees

## Viewing Child Performance

### Selecting a Child

1. From dashboard, click on child's card
2. Or use child selector dropdown

### Performance Overview

View detailed metrics:
- **Academic Average**: Overall percentage
- **Subject-wise Performance**: Grades per subject
- **Assignment Completion**: Pending vs completed
- **Attendance Rate**: Present/absent ratio
- **Recent Grades**: Latest exam results

## Attendance Tracking

### Viewing Attendance

1. Select child
2. Click **Attendance**
3. View:
   - Date-wise records
   - Status (Present/Absent/Late/Excused)
   - Subject-wise attendance
   - Overall percentage

### Attendance Alerts

Receive notifications when:
- Attendance falls below 75%
- 3 consecutive absences
- Pattern of tardiness

## Grades

### Viewing Grades

1. Select child
2. Click **Grades**
3. See all published grades:
   - Subject
   - Exam type
   - Marks obtained
   - Grade
   - Class average

### Performance Trends

- Track progress over time
- Compare with class average
- Identify improvement areas

## Fee Payments

### Viewing Fee Details

1. Select child
2. Click **Fees**
3. View:
   - Total fees
   - Paid amount
   - Pending amount
   - Due dates

### Payment History

- All past payments
- Download receipts
- Track installments

### Making Payments

1. Click **Pay Fees**
2. Select fee items
3. Proceed to payment gateway
4. Complete payment
5. Download receipt

**Note**: Payment confirmation sent via email.

## Communication with Teachers

### Sending a Message

1. Click **Communication**
2. Click **New Message**
3. Fill in:
   - Select child
   - Select teacher (or general)
   - Subject
   - Message
   - Message type (Inquiry/Concern/Feedback)
   - Priority (Low/Medium/High)
4. Click **Send**

### Message Types

- **Inquiry**: General questions
- **Concern**: Academic/behavioral issues
- **Request**: Meetings, documents
- **Feedback**: Positive feedback
- **Complaint**: Formal complaints
- **Appreciation**: Thank you messages

### Viewing Replies

- Check **Communication** section
- Messages show status:
  - Sent
  - Read
  - Replied
  - Resolved
- View teacher's response
- Continue conversation if needed

## Announcements

### Viewing Announcements

1. Click **Announcements**
2. See announcements for all children
3. Filter by:
   - Child
   - Priority
   - Type

### Announcement Types

- General notices
- Exam schedules
- Events
- Holiday notices
- Emergency alerts

## Exam Schedules

### Viewing Exams

1. Select child
2. Click **Exams**
3. View upcoming exams:
   - Subject
   - Date & time
   - Duration
   - Room number
   - Marks

### Download Schedule

- Click **Download** for PDF
- Print for child's reference

## Timetable

### Weekly Schedule

1. Select child
2. Click **Timetable**
3. View weekly class schedule:
   - Day and time
   - Subject
   - Teacher
   - Room

## Reports

### Generating Reports

1. Click **Reports**
2. Select child
3. Choose report type:
   - **Performance Report**: Grades and progress
   - **Attendance Report**: Attendance summary
   - **Fee Report**: Payment history
   - **Progress Report**: Comprehensive report
4. Select date range
5. Click **Generate**
6. Download PDF

## Multiple Children

### Managing Multiple Children

If you have multiple children:
1. Dashboard shows all children
2. Switch between children easily
3. Compare performance
4. Track each child separately

### Combined View

- View all children's attendance
- See all pending fees
- Track all communications
- Unified announcements

## Profile Management

### Updating Profile

1. Click your name
2. Select **Profile**
3. Update:
   - Contact information
   - Emergency contacts
   - Preferred contact method
4. Click **Save**

### Managing Relationships

- View linked children
- Update relationship type
- Add emergency contacts

## Notifications

### Email Notifications

Receive emails for:
- Low attendance alerts
- Poor performance warnings
- Fee due reminders
- Teacher replies
- Important announcements

### Managing Notifications

1. Go to Profile > Notifications
2. Choose which alerts to receive
3. Set frequency preferences

## Tips for Parents

### Stay Engaged

1. **Login Weekly**: Check progress regularly
2. **Monitor Attendance**: Address absences promptly
3. **Review Grades**: Discuss with child
4. **Communicate**: Reach out to teachers
5. **Pay Fees**: Avoid late payments

### Supporting Your Child

- **Encourage**: Positive reinforcement
- **Set Goals**: Academic targets
- **Schedule Study**: Regular study time
- **Limit Distractions**: Focus environment
- **Celebrate Success**: Recognize achievements

## Troubleshooting

### Can't See Child's Data
- Verify child link is active
- Contact school admin
- Check child's enrollment status

### No Reply from Teacher
- Teachers typically reply within 2-3 days
- Mark as urgent if critical
- Contact admin if no response in 1 week

### Payment Issues
- Check card details
- Try different payment method
- Contact accounts: accounts@college.edu

## Support

### Getting Help

- **Email**: parentsupport@college.edu
- **Phone**: [Parent Support Number]

- **Hours**: 9 AM - 5 PM weekdays

## Security Tips

1. **Keep Password Safe**: Don't share
2. **Log Out**: Always log out on shared devices
3. **Verify Emails**: Check sender before clicking links
4. **Report Issues**: Contact support immediately

## FAQs

**Q: How often is attendance updated?**
A: Daily, after each class.

**Q: Can I pay fees in installments?**
A: Yes, if allowed by college. Check fee structure.

**Q: How do I download report cards?**
A: Reports > Select Report Type > Generate > Download.

**Q: Can I meet teachers in person?**
A: Yes, use Communication to request meeting.

**Q: What if grades seem incorrect?**
A: Contact teacher via Communication feature.

**Q: Can I add another parent to access?**
A: Contact admin to create additional parent account.
