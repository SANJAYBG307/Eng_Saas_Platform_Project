# Teacher User Manual

## Getting Started

Welcome to the Engineering SaaS Platform Teacher Portal. This guide will help you navigate and use the system effectively.

## Logging In

1. Navigate to your college's portal (e.g., `https://yourcollege.edu`)
2. Click **Login**
3. Enter your username and password
4. You'll be directed to your Teacher Dashboard

## Dashboard Overview

The dashboard provides a quick summary of:
- **Total Students**: Students in all your sections
- **Total Assignments**: All assignments you've created
- **Pending Submissions**: Assignments awaiting grading
- **Classes Today**: Your schedule for the day

### Quick Actions
- Create Assignment
- Mark Attendance
- View Timetable
- Enter Grades

## Managing Assignments

### Creating an Assignment

1. Click **Assignments** in the sidebar
2. Click **Create New Assignment**
3. Fill in the form:
   - **Subject**: Select the subject
   - **Section**: Select the section
   - **Title**: Assignment title
   - **Description**: Detailed instructions
   - **Due Date**: Deadline for submission
   - **Max Marks**: Total marks possible
   - **Type**: Homework, Project, Quiz, etc.
4. Click **Create Assignment**

### Viewing Assignments

- Navigate to **Assignments** to see all your assignments
- Filter by:
  - Subject
  - Section
  - Status (Active/Completed)
  - Date range

### Grading Submissions

1. Click on an assignment
2. View all student submissions
3. For each submission:
   - Review the work
   - Enter marks obtained
   - Add feedback
   - Click **Save Grade**

### Tips
- Set clear deadlines
- Provide detailed instructions
- Review submissions promptly
- Give constructive feedback

## Attendance Management

### Marking Attendance

1. Click **Attendance** in sidebar
2. Click **Mark Attendance**
3. Select:
   - Date
   - Section
   - Subject (if applicable)
4. For each student, mark:
   - **Present** (P)
   - **Absent** (A)
   - **Late** (L)
   - **Excused** (E)
5. Add notes if needed
6. Click **Submit Attendance**

### Viewing Attendance Records

- View attendance for any date range
- Filter by section or student
- Export to Excel for reports

### Attendance Reports

- Generate monthly/semester reports
- View attendance percentages
- Identify students with low attendance

## Grade Management

### Entering Grades

1. Navigate to **Grades**
2. Click **Enter Grades**
3. Select:
   - Subject
   - Section
   - Exam Type (Midterm, Final, Quiz, etc.)
4. Enter marks for each student:
   - Marks obtained
   - Total marks
5. Click **Publish Grades**

**Note**: Once published, students can view their grades.

### Grade Book

View all grades you've entered:
- Filter by subject, section, exam type
- Export gradebook
- Calculate class averages

## Communication

### Messaging Students

1. Click **Communication**
2. Click **New Message**
3. Select:
   - Recipient(s): Individual or entire section
   - Subject
   - Message
4. Click **Send**

### Responding to Parents

When parents contact you:
1. Navigate to **Communication** > **Messages**
2. View parent inquiries
3. Click **Reply**
4. Enter your response
5. Click **Send Reply**

## Timetable

### Viewing Your Schedule

- Navigate to **Timetable**
- View your weekly schedule
- See classroom assignments
- Check for any changes

## Exams

### Viewing Exam Schedule

- Navigate to **Exams**
- View upcoming exams
- See exam rooms and timings
- Download schedule

### Exam Duty

If assigned exam duty:
- Check duty schedule
- Note exam room and timing
- Bring required materials

## Reports

### Generating Reports

1. Navigate to **Reports**
2. Select report type:
   - Class Performance
   - Attendance Summary
   - Grade Distribution
3. Choose parameters:
   - Section
   - Date range
   - Subject
4. Click **Generate Report**
5. Export as PDF or Excel

## Profile Management

### Updating Profile

1. Click your name (top right)
2. Select **Profile**
3. Update:
   - Contact information
   - Profile picture
   - Bio
4. Click **Save Changes**

### Changing Password

1. Go to Profile
2. Click **Change Password**
3. Enter:
   - Current password
   - New password
   - Confirm new password
4. Click **Update Password**

## Troubleshooting

### Can't Login
- Verify username/password
- Contact IT support
- Check if account is active

### Assignment Not Showing
- Check if assignment is published
- Verify section selection
- Refresh the page

### Grades Not Saving
- Check internet connection
- Ensure all required fields are filled
- Try different browser

### Students Can't See Grades
- Verify grades are published
- Check exam date is past
- Contact admin if issue persists

## Frequently Asked Questions

**Q: How do I create a recurring assignment?**
A: Create the assignment once, then use "Duplicate" for subsequent weeks.

**Q: Can I modify grades after publishing?**
A: Yes, but changes are tracked. Contact admin for major corrections.

**Q: How do I handle late submissions?**
A: You can accept late submissions manually and note the delay in feedback.

**Q: What if a student is incorrectly marked absent?**
A: Edit the attendance record within 24 hours or contact admin.

**Q: How do I export my gradebook?**
A: Go to Grades > Export, select format (Excel/PDF), and download.
